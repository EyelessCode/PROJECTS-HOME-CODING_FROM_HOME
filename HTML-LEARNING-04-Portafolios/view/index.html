FRONTEND_BACKEND

Información

Un proyecto con una historia de su creación y proceso muy interesante. Este proyecto es un Gestionador de Fichas Médicas, permite registrar, listar, eliminar y editar Pacientes; listar Centros Médicos; listar Enfermeros/as y la Toma de Signos junto con los Signos del Paciente.

Técnologías implementadas

  • HTML
  • JavaScript
  • CSS
  • TypeScript
  • NodeJS (Framework)
